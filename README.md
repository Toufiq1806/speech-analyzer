# Speech Analysis Data Pipeline

## Project Overview
This project collects a speech from a website, processes the text, and analyzes the most frequently used words.  
The final result is stored as a dataset (CSV file) that can be used for further analysis.

This is a beginner data engineering style project that demonstrates:
- Web scraping
- Text processing
- Data analysis
- Dataset generation

## Features
- Extract speech from a website
- Clean and process text
- Count word frequency
- Identify most frequently used words
- Generate a structured dataset
- Export results to CSV

BeautifulSoup
